# LeyuSports ZH Archive

## 项目简介

本仓库用于归档和发布多个独立的 HTML 页面。这些页面以中文内容为主，涵盖不同时期的静态网页资料。仓库本身不针对任何特定域名或具体网站，仅作为 HTML 文件的存档与分发点。

## 目录结构

```
.
├── pages/          # 存放独立 HTML 页面文件
├── assets/         # 页面所需的静态资源（图片、样式表、脚本等）
└── README.md       # 本说明文件
```

- `pages/`：每个子文件夹或以单独文件形式存放一个完整的 HTML 页面，文件名通常反映页面主题或归档时间。
- `assets/`：按需组织，部分页面可能将资源内嵌于 HTML 中。

## 页面归档说明

- 所有页面均为静态 HTML，无需后端支持，可直接在浏览器中打开查看。
- 页面内容可能包含旧版信息、历史资料或已不再维护的样式结构。
- 归档文件保持原始状态，不进行主动更新或内容修订，仅作历史记录用途。
- 文件命名遵循 `{描述性名称}_{日期或版本}.html` 的格式（部分文件可能无日期后缀）。

## 使用方式

1. 克隆本仓库：
   ```bash
   git clone https://github.com/your-username/leyusports-zh-archive.git
   ```
2. 进入 `pages/` 目录，双击任意 `.html` 文件即可在默认浏览器中查看。
3. 也可直接通过 GitHub Pages 或任何静态文件服务器部署。

## 维护说明

- 本仓库不定期更新，主要添加新的归档页面或修正文件链接错误。
- 不承诺更新频率，也不提供对页面内容的解释或支持。
- 欢迎通过 Issues 提交链接失效报告或建议添加的归档文件（需为合法且公开的 HTML 内容）。
- 不接受对已有页面内容的修改请求，除非涉及明显的文件损坏或安全风险。
